vfs
  .src('js/vendor/*.js', { cwd: 'src', cwdbase: true, read: false })
  .pipe(tap((file) => { (1)
    file.contents = browserify(file.relative, { basedir: 'src', detectGlobals: false }).bundle()
  }))
  .pipe(buffer()) (2)
  .pipe(uglify())
  .pipe(gulp.dest('build'))
1 The tap function is used to wiretap the data in the pipe.
2 Wrap each streaming file in a buffer so the files can be processed by uglify. Uglify can only work with buffers, not streams.

Here’s the same code with the wrap role added, which enables pre-wrap, if that’s your speed.
